# Repository Summary: `Pikul-cmu/basic_python`

This repository contains a **10-week introductory Python course** delivered entirely through Jupyter/Google Colab notebooks. The curriculum provides a strong beginner-level progression from **basic Python syntax** to **structured problem-solving and numerical data processing**.

## Course Outline

### Week 01: Python Fundamentals
* **File:** `Introduction_to_Python_Week01.ipynb`
* **Topics:** Introduction to `print('Hello World')`, string manipulation (escape sequences, multiline strings, concatenation), numeric data types, basic arithmetic, and understanding syntax/type errors.
* **Outcome:** Students can display information, work with basic strings and numbers, and understand elementary Python syntax.

### Week 02: Variables, Input, and Output
* **File:** `Introduction_to_Python_Week02.ipynb`
* **Topics:** Variable assignment, dynamic typing, type conversion (`int()`, `float()`, `str()`, `bool()`), using `input()`, and the **Input–Process–Output (IPO) pattern**.
* **Outcome:** Students can receive user input, process it using variables and expressions, and display formatted results.

### Week 03: Input–Process–Output Practice
* **File:** `Introduction_to_Python_Week03.ipynb`
* **Topics:** Review of the IPO pattern, arithmetic operator precedence, modern f-strings for formatted output, and practical calculation exercises (e.g., converting kilograms to pounds).
* **Outcome:** Students can translate a numerical problem into a complete IPO-based Python program.

### Week 04: Conditional Statements *(Content Pending)*
* **File:** `Introduction_to_Python_Week04.ipynb`
* **Topics:** Based on the course sequence, this logically covers **conditional statements**. 
* *Note: The file currently exists in the repository, but contains no readable notebook content and needs to be checked or re-uploaded.*

### Week 05: `for` Loops
* **File:** `Introduction_to_Python_Week05.ipynb`
* **Topics:** Repetition and automation using `for` loops, the `range()` function, iterating through strings, and nested loops.
* **Outcome:** Students can use counted repetition to automate tasks and process sequences.

### Week 06: Lists
* **File:** `Introduction_to_Python_Week06.ipynb`
* **Topics:** Ordered and mutable collections, indexing, slicing, list methods (`append()`, `insert()`, `remove()`), list comprehensions, and nested lists.
* **Outcome:** Students can store, access, modify, and process collections of data.

### Week 07: `while` Loops and Exception Handling
* **File:** `Introduction_to_Python_Week07.ipynb`
* **Topics:** Condition-controlled repetition, loop-control keywords (`break`, `continue`, `else`), and exception handling using `try` and `except`.
* **Outcome:** Students can implement repetition when iteration counts are unknown and handle invalid input safely.

### Week 08: Dictionaries and Sets
* **File:** `Introduction_to_Python_Week08.ipynb`
* **Topics:** **Dictionaries** (collections of key–value pairs, accessing/modifying values, iterating) and **Sets** (collections of unique values, membership checking, set operations).
* **Outcome:** Students can organize data using meaningful keys and perform operations on collections of unique values.

### Week 09: Functions
* **File:** `Introduction_to_Python_Week09.ipynb`
* **Topics:** Defining functions with `def`, parameters and arguments, variable scope, and the differences between local and global variables.
* **Outcome:** Students can divide a program into reusable, task-specific components.

### Week 10: NumPy
* **File:** `Introduction_to_Python_Week10.ipynb`
* **Topics:** Introduction to the NumPy `ndarray`, array creation, array properties (dimensions, shape), elementwise operations, broadcasting, matrix operations, and reduction operations (`sum`, `mean`).
* **Outcome:** Students can perform efficient numerical and array-based computation using NumPy.
